    > 操作

    • ↑↓←→ : 主人公（赤いハート）を動かす
    • Z : 決定 / 話す / 調べる / 会話を進める
    • Z長押し : 会話を早送り
    • X : メニューを開く / 戻る
    • C : フィールドノートを開く
    • 光るドアに近づくと次の部屋へ。クリックでも移動できる。

    > ゲームのヒント

    • 同じ人に何度も話しかけると、毎回少し違う話を聞けることがあります。
    • 雑談の中に、QCD検討会で役立つヒントが隠れていることがあります。
    • 赤いハートは、あなたの意思とメンタル余裕を表しています。
    • Zで話す/決定、Xでメニュー/戻る、Cでノートを開きます。

    > ステータス

    • 設計理解 / 試作推進 / 評価力 / 説明力 / 信頼度 / メンタル余裕
    • 相談・前提そろえ・現物確認・記録を残す選択は信頼度/説明力を上げる。
    • 無理しすぎる選択は、進捗は上がるがメンタル余裕が下がる。
